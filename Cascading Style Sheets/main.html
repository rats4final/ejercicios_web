Que es CSS?

Css significa Cascading Style Sheets u Hojas de Estilo en Cascada

Basicamente CSS sirve para estilizar el html y definir como se van a mostrar dichos elementos

Las hojas de estilo externas son almacenadas en archivos con la extension .css

Sintaxis de CSS

Una regla de css consiste de un selector y un bloque de declaracion

    h1 {color: blue; font-size:12px;}

El selector se refiere o apunta al elemento de html que queremos estilizar

El bloque de declaracion contiene una o mas declaraciones separadas por punto y coma

Cada declaracion contiene una propiedad y valor, separado por puntos dobles

Multiples declaraciones son separadas por punto y coma, y los bloques de declaracion son rodeados por llaves 

    p {
      color: red;
      text-align: center;
    }

Ejemplo

En este ejemplo todos los elementos <p> seran alineados al centro y tendran un color rojo

p es el selector, todo lo que esta dentro de {} es el bloque de declaracion, y tenemos las propiedades color y text-align con sus respectivos valores de red y center

Selectores en CSS

Un selector en css, selecciona el elemento html que quieres estilizar

Podemos dividir los selectores de CSS en 5 categorias

  • Selectores Simples(seleccionan elementos basados en nombre, id o clase)
  • Selectores Combinatorios(seleccionan elementos en base a una relacion especifica entre ellos)
  • Selectores de pseudo-clases(seleccionan elementos basados en un cierto estado)
  • Selectores de pseudo-elementos(seleccionan y estilizan una parte de un elemento)
  • Selectores de atributo(seleccionan elementos basados en un atributo o en el valor de un atributo)

El selector de elementos de CSS

El selector de elementos, selecciona elementos HTML, basandose en el nombre del elemento, o sea el nombre del tag 

      p {
        color: red;
        text-align: center;
      }

Ejemplo

Como vimos anteriormente, aqui todos los elementos seran alineados al centro, con un color rojo

El selector de id de CSS

El selector por id de CSS, usa el atributo "id" de un elemento HTML, para seleccionar un elemento especifico

El id de un elemento es unico dentro de un pagina, por lo tanto el selector de id lo usa para seleccionar un unico elemento

Para seleccionar un elemento con una id especifica, se escribe un #(hash), seguido del id del elemento

      #elemento1 {
        text-align: center;
        color: red;
      }

Elemento1

Ojo que un id no puede empezar con un numero

El selector por clase de CSS

Este selector selecciona elementos con un atributo de clase especifica

Para seleccionar elementos con una clase especifica, escribe un punto(.) seguido por el nombre de la clase 

      .center{
        text-align: center;
        color: blue
      }

Clase center

Tambien puedes especificar que solo elementos especificos de HTML deberian ser afectados por una clase 

      p.left{
        text-align: left;
        color: yellow;
      }

Solo parrafos

Ademas, los elementos HTMl pueden referirse a mas de una clase, esta es la tecnica que usan la mayoria de librerias y frameworks de css para proporcionar una experiencia rica en clases

El selector universal de CSS

Este selector, selecciona a todos los elementos HTML de una pagina

      *{
        text-align: center;
        color: blue;
      }

El selector por grupos de CSS

Este selector selecciona a todos los elementos HTML con las mismas definiciones de estilos

Por ejemplo, en el siguiente fragmento tenemos varios elementos que usan los mismos estilos 

      h1 {
        text-align: center;
        color: red;
      }
      
      h2 {
        text-align: center;
        color: red;
      }
      
      p {
        text-align: center;
        color: red;
      }

Para minimizar el codigo seria mejor agrupar a los selectores

Para agrupar selectores, separa cada selector con una coma

      h1, h2, p {
        text-align: center;
        color: red;
      }

Inclusive se pueden agrupar diferentes tipos de selectores con esta tecnica

      #car, .point, h2 {
        color:blue
      }

Como agregar CSS?

Bien ya vimos las caracteristicas principales de CSS, sin embargo aun no hemos visto como podemos agregar CSS a nuestra pagina y las distintas formas que hay para hacerlo

Existen 3 maneras de agregar CSS

  • CSS Externo
  • CSS Interno
  • CSS Dentro de linea(elemento)

CSS Externo

Con una hoja de estilo externa, puedes cambiar la apariencia de un sitio web entero con solo un archivo

Cada pagina HTML debe incluir una referencia a una hoja de estilo externa dentro del elemento <link>, dentro de la seccion head 

      <!DOCTYPE html>
      <html>
      <head>
      <link rel="stylesheet" href="mystyle.css">
      </head>
      <body>

      <h1>Este es un encabezado</h1>
      <p>Este es un parrafo</p>

      </body>
      </html>

Una hoja de estilo externa puede ser escrita en cualquier editor y debe ser guardada con la extension .css

La hoja de estilo externa no deberia contener ninguna etiqueta HTML

Asi es como se ve el archivo "mystyle.css" :

      body{
        background-color: lightblue;
      }
      h1{
        color: navy;
        margin-left: 20px;
      }

Nota:No se debe agregar un espacio entre el valor de las propiedades que tienen unidades, por eso es 20px, no 20 px

CSS Interno

Una hoja de estilo interna puede ser usada si una solo pagina de HTML tiene un estilo unico

La hoja de estilo interna es definida dentro del tag <style> dentro de la seccion head 

      <!DOCTYPE html>
      <html>
      <head>
      <style>
      body {
        background-color: linen;
      }

      h1 {
        color: maroon;
        margin-left: 40px;
      }
      </style>
      </head>
      <body>

      <h1>This is a heading</h1>
      <p>This is a paragraph.</p>

      </body>
      </html>

CSS en linea

Un estilo dentro de linea puede ser usado para aplicar un estilo unico a un solo elemento

Para usar estilos dentro de linea, agregar el atributo style al elemento relevante. El atributo style puede contener cualquier propiedad de CSS 

      <!DOCTYPE html>
      <html>
      <body>

      <h1 style="color:blue;text-align:center;">This is a heading</h1>
      <p style="color:red;">This is a paragraph.</p>

      </body>
      </html>

Usar estilos dentro de linea hace que perdamos una de las ventajas de una hoja de estilo(debido a que mezclamos contenido con presentacion). Use este metodo escasamente

Multiples hojas de estilo

Si algunas propiedades han sido definidas para el mismo selector(elemento) en diferentes hojas de estilo, el valor de la ultima hoja de estilo leida sera usada

Por ejemplo asumamos que una hoja de estilo externo tiene los siguientes estilos para el elemento <h1> 

      h1{
        color: navy;
      }

Luego asumamos que una hoja de estilo interna tambien tiene el siguiente estilo para el tag h1 

      h1{
        color: orange;
      }

Si la hoja de estilo interna es definida despues del link a la hoja externa, entonces los elementos h1 seran color naranja

Sin embargo, si la hoja de estilo interna es definida antes del link a la hoja de estilo externa, entonces los elementos h1 seran color azul marino

Orden de cascada

Que estilo sera usado cuando hay mas de un estilo especificado para un elemento HTML

Todos los estilos en una pagina se "ordenaran en forma de cascada" hacia una nueva hoja de estilo "virtual" siguiendo el siguiente orden, donde el numero 1 tiene la mayor prioridad

  1. Declaraciones con !important
  2. Estilo dentro de linea
  3. Hojas de estilo externas e internas(en la seccion head)
  4. Estilos por defecto del navegador

Colores RGB en CSS

Un color RGB representa, fuentes de luz Red, Green y Blue

En css un color puede ser especificado como un valor rgb usando la siguiente formula

rgb(red,green,blue)

Cada parametro define la intensidad del color entre 0 y 255

Por ejemplo, rgb(255,0,0) is mostrado como rojo, porque rojo esta puesto a su maximo valor (255) y los otros estan puestos a 0

Para mostrar negro todos los valores deben estar 0

Y para mostrar blanco todos los valores deben estar en 255
