Aprendiendo JavaScript

JavaScript sirve para cambiar el contenido HTML y hacerlo interactivo

JS sirve para todo lo que te puedas imaginar de desarrollo web

Puede cambiar atributos, valores, estilos, ocultar, mostrar y mucho mas

El tag script

El codigo js va dentro del tag script

En este caso el codigo ejecuta una funcion cuando ocurre un evento(clicar el boton)

El tag script puede ser puesto en cualquier lugar del documento, aunque lo recomendable es al final del body o en el head

Podemos llamar js externo con el atributo src del tag script

Declaraciones

Las declaraciones en JavaScript estan compuests de valores, operadores expresiones, comentarios y palabras clave o reservadas

Punto y coma ;

Segun tengo entendido, usar puntos y comas en JS es buena practica ya que en versiones superiores de ECMAScript el uso de estas es opcional aunque eso puede derivar en problemas a la hora de debuggear

Espacios en blanco

JavaScript ignora los espacios en blanco asi que estos pueden ser usados para darle una mayor apariencia y redactabilidad al codigo

Bloques de codigo

Lineas de codigo pueden ser agrupadas dentro de funciones, permitiendo asi ejecutarlas cuando se las necesite
