Aprendiendo JavaScript

JavaScript sirve para cambiar el contenido HTML y hacerlo interactivo

JS sirve para todo lo que te puedas imaginar de desarrollo web

Puede cambiar atributos, valores, estilos, ocultar, mostrar y mucho mas

El tag script

El codigo js va dentro del tag script

En este caso el codigo ejecuta una funcion cuando ocurre un evento(clicar el boton)

El tag script puede ser puesto en cualquier lugar del documento, aunque lo recomendable es al final del body o en el head

Podemos llamar js externo con el atributo src del tag script
