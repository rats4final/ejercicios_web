Rutas de archivos

En html existen varias formas de llamar archivos(principalmente para el atributo src)

  • img src="picture.jpg" La imagen esta dentro de la misma carpeta que el archivo html
  • img src="images/picture.jpg" La imagen esta dentro de la carpeta images, la cual esta en el mismo directorio que el html
  • img src="/images/picture.jpg" La imagen esta dentro de la carpeta images, EN EL DIRECTORIO RAIZ DEL SERVIDOR
  • img src="../picture.jpg" La imagen se encuentra una carpeta un nivel anterior a la carpeta actual del html

Las rutas absolutas apuntan a la url completa del archivo

Las rutas relativas apuntan a un archivo relativo a la pagina actual

Atributos de los forms

Los forms tienen varios atributos utiles

El atributo action, define una accion que va a ser ejecutada cuando se envie el formulario

usualmente aqui se pone una ruta, direccion de un archivo o url a donde seran enviados los datos

El atributo target, es igual al del tag anchor, ahi defininimos si al enviar el formulario se abrira otra ventana, se mostrara en el frame padre, etc

El atributo method define con que metodo http enviaremos los datos

El default es GET

y los unicos validos(en html) son el metodo GET y POST

El atributo autocomplete define si se permiten o no las sugerencias del navegador en los inputs

Ej: autocomplete="on" o autocomplete="off"

Finalmente el atributo novalidate sirve para que cuando este este presente, no se validen los inputs del formulario

Un atributo muy importante que no se menciona mucho es el enctype

Este solo se usa si tambien se esta usando el metodo post

dicho atributo especifica como deberia ser codificada la data al enviarla a x lado

Esto es importante al subir archivos, ya que debemos usar el atributo enctype + el valor:
